# Recherche : Découpage des unités de texte — Proverbes 10 à 29

**Date :** 2026-03-02
**Objectif :** Identifier les unités de texte (versets isolés vs groupes multi-versets) pour le moteur de découverte de ProverbSeed, en s'appuyant sur les commentateurs bibliques académiques.

## Conclusion principale

Le consensus académique identifie **une rupture structurelle majeure à Proverbes 22:17** :

- **10:1 – 22:16** : Collection salomonique I. Essentiellement des **versets isolés** (distiques indépendants), sans lien syntaxique entre eux. Chaque verset = une unité de texte.
- **22:17 – 24:22** : "Paroles des Sages I". Style instructif multi-verset. Chaque "parole" couvre **2 à 7 versets**.
- **24:23 – 24:34** : "Paroles des Sages II" (addendum). Unités de 1 à 5 versets.
- **25:1 – 29:27** : Collection salomonique II (compilée par les hommes d'Ézéchias). Majorité de **versets isolés**, avec quelques paires ou trios identifiables.

## SECTION A — Proverbes de Salomon I (10:1 – 22:16)

**Principe** : Chaque verset est une unité de texte indépendante (distique binaire antithétique ou synonymique).
**Référence** : Hildebrandt (1988), Fox (2009), Waltke (2004).

> ✅ **375 unités** = **375 versets individuels**
> Déjà implémentés correctement dans `proverbs.json`.

## SECTION D — Proverbes de Salomon II, Collection d'Ézéchias (25:1 – 29:27)

**Principe** : Cette collection est majoritairement composée de **versets isolés**, mais avec des groupements multi-versets reconnaissables. La structure interne est marquée par des *inclusio chiastiques* plutôt que par des superscriptions (Waltke).

**Sous-sections identifiées par les chercheurs :**
- **25:2 – 27:27** : Proverbes de comparaison ("empiriques"), style plus imagé
- **28:1 – 29:27** : Proverbes antithétiques à forte tonalité théologique/morale (33 antithèses)

## Notes sur le schéma de données

Pour les unités multi-versets, le schéma JSON doit être étendu :

```json
{
  "id": "PROV_22_17_21",
  "chapter": 22,
  "verseStart": 17,
  "verseEnd": 21,
  "text": "Prêts l'oreille et entends les paroles des sages... [texte complet]",
  "reference": "Proverbes 22:17-21",
  "isMultiVerse": true,
  "section": "sages_1",
  "sayingNumber": 0,
  "suggestions": ["Sagesse", "Instruction", "Connaissance"]
}
```

Pour les versets isolés, le schéma reste inchangé :
```json
{
  "id": "PROV_10_1",
  "chapter": 10,
  "verse": 1,
  "text": "...",
  "reference": "Proverbes 10:1",
  "suggestions": [...]
}
```
